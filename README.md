# ts-quiz-app-backend

## Summary

Welcome to the Chat Quiz App Backend repository! This backend module, developed using Node.js with Express, powers the interactive and dynamic features of the Chat Quiz App.

## Tecnologies
- Nodejs
- Typescript
- Expressjs
- MongoDB
- Redis
- Docker
- Unit Tests

### Install dependencies:

```console
$ yarn
```

### Start the server:

```console
$ yarn start
```
