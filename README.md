# ts-quiz-app-backend

## Summary

Welcome to the Chat Quiz App repository! This backend module, developed using Node.js with Express, powers the interactive and dynamic features of the Chat Quiz App.

## Tecnologies
- Nodejs
- Typescript
- Expressjs
- MongoDB
- Redis
- Docker
- Unit Tests

### Install dependencies:

```console
$ yarn
```

# Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
